### 变更

- 将 1.2.0 热门词包 UI 原型调整为更接近 Typeless 的轻玻璃风格，并加入浅色/深色切换按钮。
- 过滤内部 UI 验收使用的常用词测试数据，避免诊断词条残留在真实用户常用词列表中。
- 将高精度语音包“最新版本”文案调整为“当前内置版本”，避免把内置清单检查误表达为实时上游更新检查。
- 高精度语音包清单增加版本信息，设置页新增独立“更新状态”和“检查更新”入口，避免和“是否准备好”混在一起。
- 加强“翻译成英文 + 邮件”输出约束：保留明确收件人、按要求输出编号列表，且未要求主题时不主动添加主题行。
- 确认式常用词建议会过滤过长候选和口头结束词，避免把完整句子、私人正文或“OK / 好了 / 完成”等口头噪声加入建议。
- 将常用词建议相关文案从“记忆建议 / 记住写法”调整为“常用词建议 / 加入常用词”，避免用户误解为静默学习。
- 在产品描述中突出 AI 输出默认调用 DeepSeek V4 Flash，并说明按当前官方 API 价格和日常用量估算成本很低。

## 1.0.0 - 2026-06-24

ReadyType 首个公开候选版本。

### 新增

- GitHub Issue 模板和 README 反馈入口，覆盖安装、权限、快捷键、粘贴、识别准确率和输出语气问题。
- 中文优先的 macOS 语音输入，双击 `Option` 开始和完成。
- 语音输入中按 `Esc` 取消。
- 支持直接转文字、整理成文、翻译成英文和写给 AI 四种输出方式。
- 使用 DeepSeek 处理需要 AI 的输出，密钥存储在 macOS 钥匙串。
- 自动识别路由：优先保持快速响应，适合时使用更准确的本机识别。
- 高精度语音包下载、准备、状态展示和删除入口。
- 常用词和确认式个性化建议，用于改善术语较多的输入。
- 菜单栏浮窗、主控制台、设置、权限、新用户引导和低打扰语音输入浮层。
- 自动粘贴，失败时复制到剪贴板。
- `.app`、`.zip` 和 `.dmg` 打包脚本。

### 说明

- 1.0.0 分发包未签名、未公证。
- 语音识别不需要额外填写云端语音 API 密钥。
- AI 输出方式会把当前文本发送给 DeepSeek 处理。
