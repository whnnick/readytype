# 测试邀请文案

下面这段可以直接发给试用用户。

```text
我做了一个 macOS 语音输入工具 ReadyType，现在开放 1.0.0 公开测试版。

它主要解决这几件事：
- 双击 Option 开始说话，再双击 Option 输出到当前输入框。
- 支持直接转文字、整理成文、中文翻译成英文、整理成适合发给 AI 的任务说明。
- AI 输出默认调用 DeepSeek V4 Flash，日常语音整理和翻译的使用成本通常很低。
- 语音识别本身不需要额外填写云端语音 API 密钥。

下载地址：
https://github.com/whnnick/readytype/releases/latest

首次安装说明：
1. 下载 ReadyType.dmg。
2. 打开 DMG，把 ReadyType.app 拖到 Applications。
3. 因为当前版本未签名，首次打开时 macOS 可能提示无法验证开发者。请右键 ReadyType.app，选择“打开”，再确认“打开”。
4. 打开后按提示授权麦克风、语音识别和辅助功能。
5. 如果要使用整理成文、翻译成英文和写给 AI，需要在设置里填写自己的 DeepSeek API Key。

建议你重点试一下：
- 在微信、备忘录、浏览器、邮件或文档工具里输入。
- 短句、长句、中英混说、人名、项目名和专业词。
- 双击 Option 开始/结束，以及 Esc 取消。
- 自动粘贴是否稳定；如果没粘贴，看看剪贴板里是否已有结果。
- 输出语气是否自然，尤其是聊天、邮件、笔记和 AI 工具场景。

如果遇到问题，可以在这里反馈：
https://github.com/whnnick/readytype/issues

反馈前请删掉 API Key、私人聊天、私人邮件、真实客户信息和敏感业务内容。
```
