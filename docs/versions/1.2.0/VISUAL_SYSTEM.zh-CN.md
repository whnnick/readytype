# ReadyType 1.2.0 视觉与动效规范

## 视觉定位

系统级、克制、清晰、轻科技感。参考 Typeless 的低打扰体验和 Apple 的材质层次，但保留 ReadyType 自己的品牌颜色、图标和中文表达。

## 主题

- 跟随系统：默认。
- 浅色：浅灰画布、白色内容层、深色文字、低饱和绿色状态色。
- 深色：接近系统黑的画布、抬升的深灰内容层、高对比文字。
- 同一语义 token 在两套主题中保持一致，不在视图内散落硬编码颜色。

## HUD

- 所有状态共用 `246 x 44 pt` 黑白液态玻璃胶囊，不随主窗口外观切换；黑色玻璃主体搭配白色文字、进度和操作图标，波形仅以中间绿色作为品牌状态点缀，避免灰色浑浊感和状态间深浅跳变。
- 录音态展示真实音量波形、短状态和计时；识别与整理态展示状态和底部 Thinking 进度条。
- 进度条按识别、整理阶段推进并在真实完成前停在 100% 以下，不伪造精确百分比。
- 只用细描边、顶部高光和中性阴影与当前 App 分离；绿色仅作为 ReadyType 状态强调。
- macOS 26 使用系统原生液态玻璃，旧系统使用自适应材质降级。录音态右上角提供关闭按钮，点击与 `Esc` 进入同一取消路径；每天第一次使用短暂显示 1.6 秒“按 Esc 退出”，当天后续使用不主动提示，悬停按钮仍可查看。

## 动效状态

| 状态 | 动效 |
| --- | --- |
| 出现 | 10-12 pt 上浮，160-200 ms 淡入 |
| 正在听 | 原创双音提示后出现；波形响应实际音量 |
| 正在识别 | 外壳不变，Thinking 进度条推进到中段 |
| 正在整理 | 外壳不变，状态文字交叉淡入，进度条继续推进 |
| 已完成 | 单次绿色确认，约 900 ms 后淡出 |
| 已复制 | 琥珀色提示，约 1.5 s 后淡出 |
| 出错 | 单次轻微水平反馈，随后静止 |

## 排版和控件

- 使用系统字体，不缩放字号跟随窗口宽度。
- 主窗口标题、分组标题和辅助说明建立三级层次。
- 二元设置使用开关，多选使用菜单，外观使用三段式选择。
- 图标优先 SF Symbols；不使用文字胶囊替代常见图标按钮。
- 卡片圆角不超过 8 pt，避免卡片套卡片。

## 无障碍

- 文本与背景达到可读对比度。
- 颜色不作为唯一状态信号。
- 支持键盘导航、VoiceOver 标签和减少动态效果。
- HUD 文案在最窄尺寸下不截断关键动作。
