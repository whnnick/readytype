# ReadyType 1.2.0

ReadyType 1.2.0 聚焦 UI 与交互重构：把产品从“需要管理模式的控制面板”调整为“平时安静运行、说话时清楚反馈”的系统级语音输入体验。

## 核心方向

- 语音输入浮层是主要体验，主窗口退居设置和管理。
- 默认自动判断 App、语气和写作场景，减少用户手动选择。
- 提供跟随系统、浅色、深色三种外观。
- 参考 Typeless 的低打扰交互原则，但不复制其品牌资产或界面。
- 保留现有录音、识别、DeepSeek、粘贴、常用词和权限管线。

## 版本边界

本版本不同时引入热门词包、全新识别引擎、多模型提供商或完整历史记录。热门词包顺延到 [1.4.0](../1.4.0/README.zh-CN.md)。
