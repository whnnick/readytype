# ReadyType 1.2.0

ReadyType 1.2.0 的候选方向是“热门词包”：参考成熟输入法的分层词库和云候选思路，让电影、科技产品、体育赛事等近期高频词在用户无感的情况下提高识别和整理质量。

## 当前判断

该功能不应做成实时抓取热榜或把大量热词强塞进识别器。更稳妥的路线是：

- 本地基础词库和用户常用词保持最高优先级。
- 热门词包只作为低优先级补充候选。
- 后台自动更新，失败不影响语音输入。
- 每个词有来源、分类、权重和过期时间。
- 用户默认无感使用，但可以在设置中关闭、删除和查看更新时间。

## 发布边界

1.2.0 只做词包架构和第一批稳定词包，不承诺全网实时热词，也不引入用户输入上传、复杂云服务或多个第三方 API 配置。
