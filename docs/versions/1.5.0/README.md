# ReadyType 1.5.0

ReadyType 1.5.0 is the current release candidate. It simplifies the main-window information architecture, consolidates low-frequency configuration under Settings, and improves automatic punctuation for natural Chinese dictation.

## Documents

- [Requirements](./REQUIREMENTS.md)
- [Implementation Plan](./PLAN.md)
- [Black-box Acceptance](./BLACK_BOX_TESTS.md)

## Current Scope

- Keep Home, Usage Overview, and Common Words in primary navigation.
- Pin Settings to the bottom of the sidebar.
- Move appearance, default output, AI features, speech recognition, shortcuts, permissions and privacy, and About into Settings.
- Preserve all existing features and the current recognition and output pipelines.
- Enable system automatic punctuation for fast recognition and improve parallel-item punctuation in Polished Writing.
