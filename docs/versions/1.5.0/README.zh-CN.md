# ReadyType 1.5.0

ReadyType 1.5.0 是当前发布候选版本。本版本收敛主窗口信息架构，将低频配置集中到统一设置入口，并优化自然中文听写的自动标点。

## 文档

- [需求说明](./REQUIREMENTS.zh-CN.md)
- [实施计划](./PLAN.zh-CN.md)
- [黑盒功能检查](./BLACK_BOX_TESTS.zh-CN.md)

## 当前范围

- 主导航只保留首页、使用概览和常用词。
- 设置固定在侧边栏底部。
- 外观、默认输出、AI 功能、语音识别、快捷键、权限与隐私及关于统一收进设置。
- 不删除既有功能，不改变语音识别和输出链路。
- 极速识别启用系统自动标点，“整理成文”优化并列事项的顿号、逗号和断句。
