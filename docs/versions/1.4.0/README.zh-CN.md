# ReadyType 1.4.0

ReadyType 1.4.0 建立隐私优先的匿名产品统计，为后续中英双语版本提供可验证的使用、性能和稳定性数据。

## 文档

- [需求说明](./REQUIREMENTS.zh-CN.md)
- [匿名事件规范](./ANALYTICS_SPEC.zh-CN.md)
- [实施计划](./PLAN.zh-CN.md)

## 当前边界

- 1.4.0 不上传语音、转写、最终输出、窗口标题、常用词、剪贴板或 API Key。
- 公开源码构建默认使用 `NoopAnalyticsTracker`，不会发送统计。
- 只有官方构建显式注入统计配置后才具备发送能力。
- 用户可以在权限与隐私页面关闭匿名统计。
- 本版本先建立数据基础，不同时实现英文识别和中英混说。

## 当前进度

已完成事件白名单、设置开关、核心输入漏斗接入和 No-op 默认实现。官方 Provider、线上配置验证、README 发布说明和 1.4.0 黑盒检查仍待完成，因此当前构建不会产生远程统计数据。
