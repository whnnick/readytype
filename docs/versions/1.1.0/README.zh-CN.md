# ReadyType 1.1.0

ReadyType 1.1.0 聚焦真实输入体验的产品化收口：常用词、确认式学习建议、App 场景语气、快捷键自定义和高精度语音包状态。

- [需求说明](./REQUIREMENTS.zh-CN.md)
- [实施计划](./PLAN.zh-CN.md)

## 当前判断

1.0.0 已经包含一部分底层能力，例如常用词存储、常用词导入、确认式建议、快捷键配置和高精度语音包状态展示。1.1.0 不应重复造这些底层能力，而应把它们整理成更清楚、更稳定、更适合普通用户理解的产品体验。

基线检查：
- `swift test --filter UserVocabularyStoreTests`：9 个测试通过。
- `swift test --filter SettingsViewModelTests`：18 个测试通过。

## 开发顺序

1. 常用词库产品化收口。
2. 确认式学习建议收口。
3. App 场景语气优化。
4. 快捷键自定义体验复核。
5. 高精度语音包状态和更新提示。
6. 文档、测试和发布准备。
