## 本轮发现与处理

- build 67 的 TextEdit 真实输入发现：文档场景会无根据追加“谢谢大家”。
- 根因：非邮件文档提示词缺少对致谢、署名和收尾语的完整性约束。
- 处理：新增统一的非邮件输出完整性规则，覆盖通用、聊天、笔记、文档和英文翻译；邮件场景保持原有礼貌收尾能力。
- 复验：新增真实 DeepSeek 文档用例通过，确认不再凭空追加致谢。
- build 68 真实 TextEdit 复验通过：自动粘贴成功，技术词保留正确，未再添加“谢谢大家”。
